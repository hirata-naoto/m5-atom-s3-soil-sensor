# DFROBOT SENS0604 Modbus RTU レジスタ仕様

## センサー基本情報

- **モデル**: DFROBOT SENS0604
- **通信方式**: Modbus RTU
- **ボーレート**: 9600 bps (デフォルト)
- **パリティ**: None
- **データビット**: 8
- **ストップビット**: 1
- **デフォルトスレーブアドレス**: 1

## Modbusレジスタマップ

### 読み取り専用レジスタ（入力レジスタ - Function Code 04）

| レジスタ | アドレス | 説明 | 範囲 | 単位 |
|---------|---------|------|------|------|
| 土壌水分 | 0x0000  | 土壌体積水分量 | 0-100 | % |
| 温度     | 0x0001  | センサー周辺温度 | -10-+60 | °C |
| EC値     | 0x0002  | 電気伝導度 | 0-20 | mS/cm |
| センサルアドレス | 0x0003 | スレーブアドレス | 1-247 | - |
| ボーレート | 0x0004 | 通信ボーレート | 0-4 | - |

### ボーレート設定値

| 値 | ボーレート |
|----|----------|
| 0  | 9600 |
| 1  | 19200 |
| 2  | 38400 |
| 3  | 57600 |
| 4  | 115200 |

## データ形式

### 土壌水分と温度

16ビット符号なし整数として返される:
- **土壌水分**: 小数点第1位まで (例: 653 = 65.3%)
- **温度**: 小数点第1位まで (例: 285 = 28.5°C)

**計算例:**
```
受信値: 0x028D (653 decimal)
土壌水分 = 653 / 10 = 65.3%
```

## Modbus RTUコマンド例

### 土壌水分を読む

**リクエスト** (16進数):
```
01 03 00 00 00 01 84 0A
```

- `01`: スレーブアドレス (1)
- `03`: Function Code (読み取り保留レジスタ)
- `00 00`: レジスタアドレス (0x0000)
- `00 01`: 読み取り数 (1個)
- `84 0A`: CRC

**レスポンス** (例):
```
01 03 02 02 8D 39 44
```

- `01`: スレーブアドレス
- `03`: Function Code
- `02`: バイト数 (2)
- `02 8D`: データ (0x028D = 653 = 65.3%)
- `39 44`: CRC

### 温度を読む

**リクエスト** (16進数):
```
01 03 00 01 00 01 B5 CB
```

- `01`: スレーブアドレス
- `03`: Function Code
- `00 01`: レジスタアドレス (0x0001)
- `00 01`: 読み取り数 (1個)
- `B5 CB`: CRC

## Arduino IDEでの使用

### ArduinoModbusライブラリを使う場合

```cpp
#include <ArduinoModbus.h>

// Modbusオブジェクト初期化
ModbusRTUServer modbus;

// 土壌水分を読む
int moisture_raw = modbus.holdingRegisterRead(0x0000);
float moisture = moisture_raw / 10.0;

// 温度を読む
int temp_raw = modbus.holdingRegisterRead(0x0001);
float temperature = temp_raw / 10.0;
```

### ModbusRTUライブラリを使う場合

```cpp
#include <ModbusRTU.h>

ModbusRTU mb;

// コールバック関数
bool cbData(Modbus::ResultCode event, uint16_t transactionId, void* data) {
    // データ処理
    return true;
}

// 土壌水分を読む
mb.readInputRegisters(1, 0x0000, 1, cbData);
```

## トラブルシューティング

### CRCエラーが頻発する場合

- ボーレート設定を確認 (デフォルト: 9600 bps)
- RS485接続の安定性を確認
- 終端抵抗を追加してみる

### センサーが応答しない場合

- スレーブアドレスが正しいか確認 (デフォルト: 1)
- センサーの電源供給を確認
- RS485 A/B線が正しく接続されているか確認

### 異常な値が読める場合

- バイト順序（エンディアン）を確認
- レジスタアドレスが正しいか確認
- センサーが水に浸かっていないか確認（キャリブレーション必要）

## センサーアドレス変更方法

**セットアップモード:**

1. センサーの電源を切る
2. リセットボタン（ある場合）を押しながら電源を入れる
3. Modbusコマンドでアドレスを設定:

```
Function Code: 06 (Write Single Register)
Address: 0x0003 (センサーアドレスレジスタ)
Value: 新しいアドレス (1-247)
```

4. センサーを再起動

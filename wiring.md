# 配線図と接続方法

## M5 Atom S3のピン配置

M5 Atom S3の重要なピン:

| ピン | GPIO | 機能 | 説明 |
|------|------|------|------|
| 5 | GPIO 5 | TX | UART送信 |
| 6 | GPIO 6 | RX | UART受信 |
| GPIO 1-10 | - | デジタルI/O | カスタム使用可 |
| 5V | - | 電源 | RS485モジュール電源 |
| GND | - | グラウンド | RS485モジュルグラウンド |

## RS485接続図

```
M5 Atom S3                  RS485モジュール          SENS0604センサー
==============              ==============           ================

GPIO 5 (TX)  --------+----> DI (Data In)
GPIO 6 (RX)  --------+----> RO (Data Out)
5V           --------+----> VCC
GND          --------+----> GND
                             A --------+----> A (白)
                             B --------+----> B (緑)
                                        GND ---> GND (黒)
```

## 詳細配線手順

### 1. M5 Atom S3 ↔ RS485モジュール

| M5 Atom S3 | GPIO | RS485モジュール |
|-----------|------|----------------|
| ピン 5    | 5    | DI (Data In)    |
| ピン 6    | 6    | RO (Data Out)   |
| 5V        | -    | VCC             |
| GND       | -    | GND             |

**配線例（ジャンパーワイヤー使用）:**
```
M5 Atom S3 (側面)
[ピン 5]  ← TX用ジャンパーワイヤー
[ピン 6]  ← RX用ジャンパーワイヤー
[5V]      ← 電源用ジャンパーワイヤー（赤）
[GND]     ← グラウンド用ジャンパーワイヤー（黒）
        ↓
    RS485モジュール
    [DI]  ← ピン5接続
    [RO]  ← ピン6接続
    [VCC] ← 5V接続
    [GND] ← GND接続
        ↓
    SENS0604センサー
```

### 2. RS485モジュール ↔ SENS0604センサー

SENS0604はツイストペアケーブル接続:

| RS485モジュール | SENS0604センサー | ケーブル色 |
|----------------|-----------------|----------|
| A              | A               | 黄 (Yellow) |
| B              | B               | 青 (Blue) |
| VCC            | VCC             | 茶 (Blown) |
| GND            | GND             | 黒 (Black) |

**注意**: ツイストペアケーブルのA/B線は逆に接続しないこと。

## RS485終端抵抗について

RS485は長いケーブルを使う場合、終端抵抗が必要になる場合があります。

**短距離（1m以内）**: 不要
**長距離（1m以上）**: 推奨
- 値: 120Ω
- 配置: センサー側のA-B間に接続

```
SENS0604
  A ----[120Ω]---- B
```

## LED表示（M5 Atom S3）

- **青LED点灯**: 初期化中
- **緑LED点灯**: センサーデータ正常に読み込み
- **赤LED点灯**: センサー接続エラー
- **黄LED点灯**: ボタン押下
- **消灯**: 通常状態

## ディスプレイ表示

M5 Atom S3の内蔵画面に以下の情報を表示:

```
=== Soil Sensor ===

Moisture: 65.3 %
Temp: 28.5 C
EC: 1.2 mS/cm
[OK]

Last: 123
```

- **Moisture**: 土壌水分値（%）
- **Temp**: 温度（°C）
- **EC**: 電気伝導度（mS/cm）
- **[OK]**: ステータス（成功時は緑、エラー時は赤）
- **Last**: 最後の計測からの経過時間（秒）

## 電源供給

**推奨電源:**
- M5 Atom S3: USB-C 5V / 1A以上
- RS485モジュール: M5 Atom S3からの5V供給で十分

**注意**: 土壌センサーは屋外使用のため、ノイズ対策が重要です。
- 電源ケーブルは可能な限り短くする
- ツイストペアケーブルを使用する

## トラブルシューティング

### 通信ができない場合

1. **配線を確認**
   - TX(5)/RX(6)が逆になっていないか
   - RS485 A/B線が逆になっていないか
   - 接触不良がないか

2. **ボーレート確認**
   - SENS0604のデフォルト: 9600 bps
   - コード側の `SERIAL_BAUD` が 9600 に設定されているか確認

3. **センサーアドレス確認**
   - デフォルト: 1
   - コード側の `SENSOR_ADDRESS` が正しいか確認

### 画面に表示されない場合

- M5 Atom S3のディスプレイケーブル接続を確認
- M5Unifiedライブラリが正しくインストールされているか確認

### ノイズが多い場合

- シールド付きケーブルを使用
- 5V電源のコンデンサ（100µF）を追加
- 終端抵抗（120Ω）を追加

### Modbusエラーが頻発する場合

1. **ボーレート不一致**
   - M5側: `SERIAL_BAUD` 確認
   - センサー側: Modbusレジスタ 0x0004 確認

2. **RS485接続の安定性**
   - ケーブルの接触確認
   - 終端抵抗の追加
